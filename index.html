команды Git

  • git remote add origin git@github.com:andreymar4encko26/glo.git git branch -M main git push -u origin main
  • git init — создает репозиторий в текущей папке
  • ggit commit -m «Name of commit» — зафиксировать в коммите проиндексированные изменения (закоммитить), добавить сообщение
  • git checkout -b имя ветки - добавляет ветку
  • git checkout имя ветки -меняет ветку
  • git push origin имя ветки пушит ветку в репозиторий
  • git add . — добавить все файлы в отслеживаемые
  • git log — посмотреть историю коммитов
  • git status — статус коммита
  • git pull origin master — влить изменения с удалённого репозитория (только указанная ветка)
